# Kaggle Titanic Competition

This workspace contains code and resources for solving Kaggle’s Titanic competition.

Competition overview: https://www.kaggle.com/competitions/titanic/overview

## Environment Setup

To update the environment after changes to `environment.yml`, run:

```bash
mamba env update -f environment.yml --prune
```

Activate the environment:

```bash
mamba activate kaggle
```
